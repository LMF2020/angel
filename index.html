angel直销管理系统v2.0.

本系统是个基于web开发的计算系统,系统主要包含会员管理、会员升级管理、会员业绩计算、业绩网络图展现、导出等功能 界面采用easyui,后台部分使用springMVC框架,数据库采用MySql,请使用以下命令check代码:

$ git clone origin

系统主要功能概述

会员管理

提供会员的增删改查.

会员升级管理

会员会根据每月的订单情况.系统根据doc文档中的规则自动计算会员的升级条件并作出统计

会员业绩计算

会员业绩计算分为会员(PPV)、会员(APPV)、会员(PV/BV)、会员(TNPV)等组成

图形展现

网络结构图
